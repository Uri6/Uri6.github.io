Genius Bot

About

Genius Bot is an extension with a lot of helpful automations & tools for genius.com users.

Automated actions for Genius, such as:

  • Add credits & tags to each song in an album with one link
  • Autofill Youtube/SoundCloud links via linking a playlist
  • Auto-link artwork button

Tools to improve productivity, such as:

  • Colorize brackets in the lyrics edit page
  • Reply to suggestion button
  • Alerts for missing bios & important metadata for each song on an album page
And much more.
